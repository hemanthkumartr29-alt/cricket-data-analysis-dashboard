# 🏏 Cricket Data Analysis Dashboard

## 📊 Dashboard Preview
![Dashboard](dashboard.png)

---

## 📌 Overview
This project analyzes cricket match data using SQL and Power BI to generate insights on team performance, player statistics, and match trends.

---

## 🎯 Key Features
- Team performance analysis  
- Venue-based insights  
- Top batsman & bowler tracking  
- Match trend analysis  
- Interactive filters  

---

## 🛠️ Tools Used
- SQL  
- Power BI  

---

## 📈 KPIs
- Total Runs  
- Total Wickets  
- Total Matches  

---

## 💡 Insights
- RCB scored highest total runs  
- MI has highest wickets  
- Bangalore is high scoring venue
